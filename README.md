# GrassCloud
随便玩玩，微服务构建的企业协同系统

所用技术：

- Spring Cloud
- Spring Boot
- Mybatis-Plus
- Redis
- RabbitMQ

模块介绍：

- Common：公共模块

- EurekaServer：注册中心
- ConfigServer：配置中心
- GatewayServer：网关
- GeneratorServer：代码生成
- MonitorServer：监控
- OauthServer：授权认证中心
- BaseServer：用户服务，即整个微服务中最核心的服务
